# Video Script: Installing ReviPrompt Lab Add-ins
**Duration: 5 minutes**
**Target: Revit users, all skill levels**

## Video Structure

### Intro (0:00 - 0:30)
**[Screen: ReviPrompt Lab logo/website]**

"Hi, I'm [Your Name] from ReviPrompt Lab. In this video, I'll show you exactly how to install our professional Revit automation add-ins in under 5 minutes. Whether you're using Revit 2024, 2025, or 2026, this step-by-step guide will get you up and running quickly."

**[Screen: Show final result - Revit with add-ins working]**

"By the end of this video, you'll have our tools installed and ready to transform your 4-hour manual tasks into 5-minute automated processes."

### Prerequisites (0:30 - 1:00)
**[Screen: System requirements checklist]**

"Before we start, make sure you have:
- Revit 2024, 2025, or 2026 installed
- Administrator rights on your computer
- Your ReviPrompt Lab download files ready
- About 5 minutes of time"

**[Screen: Download email example]**

"If you just purchased, check your email for the download links. If you need to re-download, visit revipromptlab.com/download."

### Step 1: Download and Extract (1:00 - 1:45)
**[Screen: Email with download links]**

"First, click your download link from the email. This will download a ZIP file to your computer."

**[Screen: File explorer showing download]**

"Navigate to your Downloads folder and find the ReviPrompt Lab ZIP file. Right-click and select 'Extract All'."

**[Screen: Extraction process]**

"Choose a location you'll remember - I recommend creating a folder like 'C:\ReviPrompt Lab' to keep everything organized."

### Step 2: Close Revit (1:45 - 2:00)
**[Screen: Revit application running]**

"Critical step: Close all Revit applications completely. The installation won't work properly if Revit is running."

**[Screen: Task Manager or Process view]**

"Check your task manager to make sure no Revit processes are running in the background."

### Step 3: Run Installer (2:00 - 3:00)
**[Screen: Extracted folder contents]**

"In your extracted folder, you'll see the ReviPrompt Lab installer. Right-click on it and select 'Run as Administrator'."

**[Screen: UAC prompt]**

"Click 'Yes' when Windows asks for permission."

**[Screen: Installer welcome screen]**

"The installer will open. Click 'Next' to continue."

**[Screen: Revit version selection]**

"Important: Select your Revit version. You can select multiple versions if you have them installed. The installer will automatically detect your Revit installations."

**[Screen: Installation progress]**

"Click 'Install' and wait for the process to complete. This usually takes 30-60 seconds."

### Step 4: Verify Installation (3:00 - 4:15)
**[Screen: Starting Revit]**

"Now let's verify everything installed correctly. Launch Revit."

**[Screen: Revit Add-ins tab]**

"Once Revit loads, look for the 'Add-ins' tab in the ribbon. You should see ReviPrompt Lab tools here."

**[Screen: ReviPrompt Lab panel]**

"Click on our panel to expand it. You'll see all your purchased tools ready to use."

**[Screen: Quick tool demonstration]**

"Let's test one quickly. I'll click on 'Sheet Renamer' and you can see it opens our professional interface."

### Step 5: Licensing and Activation (4:15 - 4:45)
**[Screen: License dialog if applicable]**

"Some tools may ask for license information on first use. Enter the license key from your purchase email."

**[Screen: Successful activation]**

"Once activated, you're ready to start automating your Revit workflows!"

### Troubleshooting (4:45 - 5:00)
**[Screen: Common issues list]**

"If you don't see the tools:
1. Make sure you ran the installer as Administrator
2. Verify you selected the correct Revit version
3. Restart Revit completely
4. Check our documentation at revipromptlab.com/docs"
