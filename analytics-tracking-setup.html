Analytics & Tracking Setup Guide

Google Analytics 4 (GA4) Setup

Step 1: Create GA4 Property

  1. Go to Google Analytics
  2. Create new property: "ReviPromptLab"
  3. Set timezone to your location
  4. Enable Enhanced Measurement
  5. Copy your Measurement ID (G-XXXXXXXXXX)

Step 2: Add to Squarespace

Add this code to your Squarespace site header:

<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX', {
    'custom_map': {
      'custom_parameter_1': 'product_tier',
      'custom_parameter_2': 'license_key'
    }
  });
</script>

Step 3: Enhanced E-commerce Tracking

Add this to track purchases and conversions:

<script>
// Track purchase events
function trackPurchase(productTier, value, transactionId) {
  gtag('event', 'purchase', {
    'transaction_id': transactionId,
    'value': value,
    'currency': 'USD',
    'items': [{
      'item_id': productTier,
      'item_name': 'ReviPrompt Lab ' + productTier + ' Pack',
      'category': 'Digital Product',
      'quantity': 1,
      'price': value
    }]
  });
}

// Track email signups
function trackEmailSignup(email) {
  gtag('event', 'sign_up', {
    'method': 'email',
    'custom_parameter_1': 'free_tier'
  });
}

// Track video views
function trackVideoView(videoTitle, duration) {
  gtag('event', 'video_play', {
    'video_title': videoTitle,
    'video_duration': duration
  });
}
</script>
