## Video Production Guidelines

### Technical Requirements
- **Resolution:** 1920x1080 (Full HD)
- **Frame Rate:** 30fps
- **Duration:** 3-4 minutes per pack, 2 minutes for overview
- **Format:** MP4 (H.264)

### Screen Recording Setup
- **Revit Version:** Use Revit 2024 for consistency
- **Sample Models:** Create representative MEP and architectural models
- **Screen Region:** Focus on Revit interface, minimize distractions
- **Cursor:** Use cursor highlighting for clarity

### Audio Guidelines
- **Microphone:** Clear, professional audio quality
- **Pace:** Moderate speed, allow time to see actions
- **Tone:** Professional but approachable
- **Background:** No background music during demonstrations

### Visual Guidelines
- **Branding:** Consistent ReviPrompt Lab branding
- **Text Overlays:** Clear, readable fonts
- **Transitions:** Smooth cuts between tools
- **Highlights:** Use callouts to emphasize key features

### Model Preparation
**MEP Model Requirements:**
- Mix of mechanical, electrical, plumbing equipment
- Some untagged equipment for before/after
- Realistic electrical loads for calculation demo
- Some connectivity issues for validation demo

**QC Model Requirements:**
- Various element types and categories
- Some quality issues (warnings, unused families)
- Multiple levels and worksets
- Mix of complete and incomplete documentation

- [ ] Clear audio throughout
- [ ] Smooth transitions between tools

- [ ] Demonstrations are easy to follow
- [ ] Call-to-action is clear
