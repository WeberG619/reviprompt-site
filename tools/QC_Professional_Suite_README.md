## Installation Instructions

### System Requirements
- Autodesk Revit 2024, 2025, or 2026
- Windows 10/11
- .NET Framework 4.8
- Microsoft Excel (for reports)

### Installation Steps

1. **Download the QC Professional Suite**
   - Extract all files to a temporary folder
   - You should have 3 tool folders: ModelAuditor, ElementCounter, DeliverableChecker

2. **Install Each Tool**
   
   For **ModelAuditor**:
   - Copy `ModelAuditor.dll` to: `%AppData%\Autodesk\Revit\Addins\2024\`
   - Copy `ModelAuditor.addin` to: `%AppData%\Autodesk\Revit\Addins\2024\`
   
   For **ElementCounter**:
   - Copy `ElementCounter.dll` to: `%AppData%\Autodesk\Revit\Addins\2024\`
   - Copy `ElementCounter.addin` to: `%AppData%\Autodesk\Revit\Addins\2024\`
   
   For **DeliverableChecker**:
   - Copy `DeliverableChecker.dll` to: `%AppData%\Autodesk\Revit\Addins\2024\`
   - Copy `DeliverableChecker.addin` to: `%AppData%\Autodesk\Revit\Addins\2024\`

3. **For Revit 2025/2026**
   - Replace `2024` with `2025` or `2026` in the paths above
   - All tools are compatible across Revit versions

4. **Restart Revit**
   - The tools will appear in the Add-Ins ribbon tab
   - Look for "Model Auditor", "Element Counter", and "Deliverable Checker"

### Verification
Open Revit and check the Add-Ins tab. You should see three new buttons:
- **Model Auditor** - Comprehensive model health check
- **Element Counter** - Advanced element counting and analysis
- **Deliverable Checker** - Project delivery readiness assessment

## Quick Start Guide

### Model Auditor
1. Open a Revit model
2. Click "Model Auditor" in the Add-Ins tab
3. Select audit categories (warnings, links, families, performance, etc.)
4. Choose auto-fix options if desired
5. Review comprehensive health report and recommendations

### Element Counter
1. Open a Revit model with elements
2. Click "Element Counter" in the Add-Ins tab
3. Configure counting options (scope, breakdowns, analysis)
4. Generate detailed element count report
5. Review categories, levels, and statistical analysis

### Deliverable Checker
1. Open a project model ready for delivery
2. Click "Deliverable Checker" in the Add-Ins tab
3. Select deliverable categories to validate
4. Review delivery readiness score and critical issues
5. Address action items before project delivery
