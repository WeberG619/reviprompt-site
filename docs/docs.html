Installation Guide

System Requirements

  • Autodesk Revit 2024, 2025, or 2026
  • Windows 10 or 11 (64-bit)
  • .NET Framework 4.8 or higher
  • Administrator privileges for installation

Installation Steps

  1. 1 Download the add-in package from your purchase confirmation email
  2. 2 Close all Revit applications
  3. 3 Run the installer as Administrator
  4. 4 Select your Revit version(s)
  5. 5 Launch Revit and verify tools appear in the Add-ins tab

Sheet Setup Starter

SheetRenamer Tool

Batch rename sheets using Excel import/export functionality. Supports all Revit project types.

Features:

  • Export current sheet list to Excel
  • Edit sheet numbers and names in Excel
  • Import changes back to Revit with validation
  • Process 100+ sheets in seconds

Frequently Asked Questions

Can I use these tools on multiple computers?

Standard licenses are for single users. Enterprise licenses support up to 10 users. See our license terms for details.

Do you support older Revit versions?

Current tools support Revit 2024-2026. For older versions, contact our support team for compatibility options.

How do I get updates?

Updates are delivered via email. Mini-packs include 6 months of updates, Professional packs include 1 year, Enterprise includes lifetime updates.

Can I customize the tools?

Yes! All products include full C# source code for customization. Enterprise customers get additional support for custom development.

Need additional help? Our support team is here to assist you.
